📚 PageTurner – Bookstore Web Application

📖 Overview

PageTurner is a modern and responsive bookstore web application developed as part of an academic project to demonstrate the integration of **Human-Computer Interaction (HCI) principles** within a **Mini Software Development Life Cycle (SDLC)**.

The application allows users to browse a large catalog of books, view detailed previews, add books to a shopping cart, and simulate a checkout process.

---

🎯 Project Objective

The goal of this project is to design a **user-friendly bookstore interface** that applies usability principles and provides an intuitive browsing and purchasing experience.

This project demonstrates:

* Mini SDLC implementation
* User-centered design
* Interactive UI/UX
* Basic e-commerce workflow

🧠 HCI Principles Applied
The project integrates usability concepts including:

* Visibility of system status
* Consistency and standards
* User control and freedom
* Error prevention
* Recognition rather than recall
* Flexibility and efficiency
* Aesthetic and minimalist design

These principles help ensure the system remains **easy to understand and navigate for users.


🔄 Mini SDLC Implementation

1️⃣ Planning

Defined the objective of creating an interactive bookstore interface.

2️⃣ Requirement Analysis
Identified key features such as:
* browsing books
* viewing previews
* cart functionality
* checkout process

3️⃣ Design
Created wireframes and prototypes for:
* homepage
* book catalog
* preview page
* checkout page

4️⃣ Implementation
Developed the website using HTML, CSS, and JavaScript.

5️⃣ Evaluation
Tested usability, navigation flow, and responsiveness.
